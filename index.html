Projects

RC Car

I designed and built a Bluetooth-controlled RC car using an Arduino Nano as the brain of the system. The project includes an HC-05 Bluetooth module for wireless communication, an H-bridge motor driver for precise motor control, an SG90 servo motor for steering, and a buck converter for efficient power regulation. Powered by dual 7.4 V LiPo batteries, the system demonstrates practical applications of embedded systems, circuit design, and motor control.

Safe Security System

I have developed a smart security system using an Arduino Uno that integrates multiple sensors and interactive components to enhance safety. The system features an LCD display, a keypad for secure password entry, a PIR motion detector to sense movement, and an ultrasonic sensor for precise distance-based intrusion detection. An SG90 servo motor and a photoresistor are used for the locking mechanism. This project demonstrates my embedded systems, circuit integration, and practical solutions to everyday security challenges.
